# Beauty Product Catalogue Frontend

This React app consumes the backend API to list, search, and view beauty products.

## Environment

Create a `.env` file in this folder with:

```
REACT_APP_API_BASE=http://localhost:3001
```

Ensure the backend is running and exposes the API at `${REACT_APP_API_BASE}/api/`.

## Features

- Top navigation with search
- Responsive grid of product cards
- Modal product detail view
- Loading and error handling
- Ocean Professional theme (blue primary, amber accents)

## Scripts

- npm start
- npm run build
- npm test
